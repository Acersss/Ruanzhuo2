# api_v1.2 (更新中)     
params:前端传给后端
response:后端返回数据
## 1. 人员管理
**/list --人员列表**  #finsh by superlgc
*根据传参查询数据并分页返回(一页7条)*
```javascript
params:{
         department, //部门
         degree,    //学历
         search,    //姓名
         time,       //签约时间
         etime,      //离职时间
         currentPage //当前页数
}
response:{
	data:[
		wid,
		name,
		sex,
		degree,
		department,
		lphone,
		state,
		signingTime,
		departureTime
	],
	totalPage,   //数据总条数
}
```
